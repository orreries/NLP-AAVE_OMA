About this project:

Hi, I'm Oriel and this project is interrogating the usage of AAVE within popular media. It does so using NLP Semantic Similarity techniques.

Using Sentence Transformers it implements the all-MiniLM-L12-v2 model to calculate cosine similarity between the comedic movie scripts and a corpus of Black Twitter posts.

Reflections & Findings

Reflection

I first applied an n-gram dictionary matching procedure to see if there was a demonstrable amount of AAVE usage in the selected films. Using a cleaned AAVE glossary, each script was evaluated for single-word and multi-word AAVE expressions.

Semantic Similarity Results (Cosine Similarity)

Across all films, the semantic similarity scores cluster tightly together, regardless of whether the film uses noticeable AAVE features.

High Similarity Cluster (~0.20–0.23)

  • Next Friday – 0.2208
  • Bridesmaids – 0.2217
  • The Hangover – 0.2339
  • Superbad – 0.2013

Why the Model Behaves This Way

The SentenceTransformer MiniLM model primarily encodes topical similarity, emotional tone, and discourse structure. What it does not encode well includes vernacular features, racialized linguistic identity, code-switching, cultural context, and sociolinguistic register.

Implications: Structural Erasure of AAVE in Machine Learning Models

  1. AAVE appears invisible in semantic embedding space. Even when a script contains unmistakable AAVE terms and Black cultural rhetoric, the model collapses those linguistic features into generic English structures.
  2. Topic overrides dialect. The embeddings capture what is being discussed, not how it is expressed.
  3. AAVE is underrepresented in training data. MiniLM was trained on standard and majority-coded English corpora.

What These Results Demonstrate

General-purpose NLP models can detect thematic similarity across texts but fail to represent or distinguish the linguistic, cultural, and racialized nuances encoded in AAVE. Machine learning systems reproduce existing linguistic hierarchies by not accounting for diverse English dialects.
